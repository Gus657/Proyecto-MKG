Qué programamos?
Quiéres saber?

Aquí podrás encontrar algunos ejemplos de programas que realizamos a lo largo del curso de Cinta Blanca. Esperamos que te sean utiles y puedas aprender sobre ellos.

Scripts

Utilizamos archivos de tipo "Script" para construir la logica de nuestra pagina web. Estos archivos nos permiten agrupar funciones (En este caso de JavaScript) para poder realizar llamadas a las funciones que se utilizaran más adelante.

Funciones

Utilizamos funciones para crear porciones de código las cuales realizan funciones especificas, al ser estas funciones tener un nombre nos permitirá llamarlas desde distintas partes de la página.

Variables

Las variables no permitirán almacenar valores que utilizaremos a lo largo de los programas que hemos desarrollado, dandoles una nueva posibilidad de funcionamiento, evitando que estos valores sea estaticos.
